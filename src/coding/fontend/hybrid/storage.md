* [[IndexedDB][indexeddb]] -- rudimentary SQL

* [[Cache Storage][cacheapi]] --  store `Request` and `Response` for `ServiceWorker` offline
* [[Persist Storage][persistent-storage]]

    * UserDefaults on iOS
    * LocalStorage on PWA
* [[Cordova][cordova-sqlite]]
